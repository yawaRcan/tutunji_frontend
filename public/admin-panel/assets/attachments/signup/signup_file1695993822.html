Installation


Hi, guys! Please read carefully. Installation is very easy process.

If you use GoDadday, Siteground or HostGator hosting you have to change some settings

You have to increase increase upload_max_filesize, php_limit_time, post_max_size before DEMO Data Install

Read my article here: https://velikorodnov.ticksy.com/article/3967/



- Step 1: Go Appearance--> Themes --> Add New Theme --> Upload Theme. Install and activate jogasana.zip.

- Step 2: Go Appearance --> Install Plugins. Install and Activate plugins.

- Step 3: Go Appearance --> Theme Settings --> Demo Install Click Import Demo. It takes ~ 5 minutes. Please wait. If you didn't finish import, repeat procedure or use Alternative Method

- Step 4: Go Settings --> Reading --> Front Page, choose needed Front page.


Alternative Method
Remember that your site database will reset entirely after using this method.

Please follow steps listed below to import demo.

- Step 1: Install and activate All-in-One WP Migration plugin to use this method.

- Step 2: Download Demo backup file. (send me request via ticket system https://velikorodnov.ticksy.com/)

- Step 3: Upload backup file to wp-content/ai1wm-backups folder.

- Step 4: Navigate to All-in-One WP Migration > Backups on admin sidebar

- Step 5: Click Restore button and wait for few seconds. You will see success message after click continue.

- Step 6: Login to the site with account: Username: admin, Password: ***** (ask me via my support area). After login with this account, you can change username and password back to your credentials.

Theme settings

Here you can find all Global settings. Global settings are applied from Theme settings if you use Default value on Page Settings. Each page has local settings and you can overwrite Global value by Local.

For example, you want to use Header 1 on all pages excerpt Blog page. You choose Header 1 in Theme settings and open Blog page and choose Header 2. In this case Header 2 will change Header 1 from Theme settings
